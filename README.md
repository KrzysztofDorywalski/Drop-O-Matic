# 💧 Drop-O-Matic

**Drop-O-Matic** is a lightweight, easy-to-use, open-source Python tool for determining dynamic contact angles in Sessile Drop and Captive Bubble experiments. 

Designed for researchers who need a fast, reliable, and transparent way to process video sequences of wetting phenomena, especially in challenging optical conditions.

---

## 🚀 Key Features

* **Dual Mode Support:** Analyzes both **Sessile Drop** and **Captive Bubble** setups.
* **Dynamic Measurement:** Specifically built to determine advancing and receding contact angles from video frames.
* **Auto-Detection:** Intelligent contour detection (OpenCV-based) that ignores substrate reflections and noise.
* **Flexible Fitting:** Toggle between **Ellipse** and **Circle** fitting models depending on droplet symmetry.
* **Export Ready:** One-click data export to `.csv` and high-resolution `.png` screenshots with overlays.
